# 概览

轻量应用云主机（ULightHost），是一种轻量级云服务器产品，专门针对中小企业和开发者提供便捷高效的云端构建服务。它比普通云服务器更加简单易用，以套餐形式整体售卖云资源，并提供高带宽流量包。用户可以一键构建应用，无需复杂的配置过程，让上云变得更加简单。
