# 云上OpenClaw(Clawdbot)快速接入QQ指南
## 概述
本文主要介绍在UCloud轻量应用云主机中部署完成OpenClaw后如何配置接入QQ 机器人。

当前文档仅适配镜像版本2026.2.17及以上版本，您可以在UCloud控制台轻量应用云主机详情页进行版本确认。

## 前置准备工作
在正式开始为OpenClaw(Clawdbot)配置接入QQ之前，请您依次检查如下事项是否准备完成：
1. 您已经拥有一个完成实名认证的腾讯 QQ 账号。
2. 您已经购买预装OpenClaw且镜像版本2026.2.17及以上版本的UCloud轻量应用云主机。

## 接入QQ

### 注册并认证 QQ 开放平台
1. 前往[QQ 开放平台](https://open.tencent.com/)，无法直接使用 QQ 账号登录，需点击注册，输入邮箱创建账号（一个邮箱仅能注册一个账号，建议使用公共 / 企业邮箱），设置登录密码（8 位及以上，含数字、字母，区分大小写）。
2. 首次注册后，按照指引设置超级管理员，填写姓名、身份证号、手机号并完成验证码验证，输入超级管理员 QQ 号，使用该手机号绑定的手机 QQ 扫码验证，验证后该 QQ 号成为账号最高权限管理员。
3. 选择主体类型（本文以 “个人” 为例），填写主体信息并完成人脸认证：使用超级管理员姓名绑定的手机 QQ 扫码进行人脸识别，审核通过后即可完成 QQ 开放平台入驻。   
![image](/images/tencentqq.png)

### 创建 QQBot 机器人 
1. 登录 QQ 开放平台，进入机器人板块，点击创建机器人（单账号可创建多个机器人）。
2. 填写机器人信息：
   名称：4-30 个字符（1 个中文字 = 2 个字符），需先检测名称可用性；
   头像：上传 png/bmp/jpeg/jpg 格式图片，无政治敏感与色情内容；
   描述：限 120 个字；
3. 填写完成后点击确认，即可完成 QQBot 机器人创建，进入机器人管理页面。

### 获取 AppID 和 AppSecret
1. 在 QQ 机器人管理页面的开发管理板块，找到AppID (机器人 ID)和AppSecret (机器人密钥)。
2. 分别点击复制按钮，将两个参数保存至记事本 / 备忘录（注意数据安全，勿泄露）；AppSecret 不支持明文保存，首次查看或遗忘需重新生成。
**注意**：此处创建的 QQ 机器人无需进行发布上架操作。


### 配置 IP 白名单
1. 前往UCloud轻量应用云主机产品控制台，复制部署 OpenClaw 的轻量应用云主机公网 IP 地址（注意保护，勿对外暴露）。
2. 返回 QQ 机器人开发管理页面，在IP 白名单输入框中添加上述公网 IP，点击保存。
提示：提审及上线前 IP 白名单不可为空；若与机器人对话提示 “该机器人去火星了，稍后再试吧”，需优先排查 IP 白名单配置是否正确。


### 沙箱配置
在 QQ 机器人开发管理的沙箱配置板块，根据使用场景选择配置类型，2026 年 1 月 31 日后 QQ 开放平台暂不支持 QQ 群配置，仅支持单独聊天配置，操作如下：
1. 选择在消息列表配置（单独聊天场景）；
2. 点击添加成员，设置成员角色（管理员 / 普通成员），添加后使用该成员的 QQ 扫码确认添加机器人；
3. 扫码成功后，该成员的 QQ 消息列表中将出现该机器人节点，机器人处于沙箱测试环境。
**可选**：可继续添加更多成员，成员均需通过 QQ 扫码完成机器人添加。


## 为OpenClaw配置模型和通道
### 应用管理
选择相应的轻量应用云主机并进入详情页，点击**应用管理**即可对AIAgent进行配置。
![image](/images/appconf.png)

### 模型配置
为OpenClaw配置模型API Key，[UModelVerse](https://docs.ucloud.cn/modelverse/README) 旨在为客户提供快速搭建 AGI 应用的能力。仅需一个 API Key，即可轻松接入 OpenAI、Gemini 兼容的 API 接口，快速构建您的专属 AGI 应用，[ModelVerse 控制台](https://console.ucloud.cn/modelverse/experience/api-keys)

### 通道配置（配置 QQ 机器人 AppID 和 AppSecret）
1. 在应用管理的通道配置输入框中，输入此前在 QQ 开放平台获取的QQ 机器人 AppID和AppSecret；
2. 点击应用并确认操作，等待几十秒完成配置；
3. 等待几十秒后查看状态，确保你的QQ通道显示**运行中**。



## 与 QQ 机器人进行交互
模型配置和通道配置完成后，即可在 QQ 端与接入 OpenClaw 的 QQ 机器人进行互动，**目前仅支持单独聊天场景**（群聊配置暂被 QQ 开放平台限制）。
