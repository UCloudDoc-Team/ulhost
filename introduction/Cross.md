# 跨境电商专区

为了助力用户拓展跨境电商、社媒营销等业务，UCloud优选多个国家的本地网络运营商，提供静态独享IP，避免因为IP问题导致流量被限等情况，防范未知风险，保障用户业务的正常进行。

## 与常规专区的区别

| 特性 | 常规专区 | 跨境电商专区 |
| :--- | :--- | :--- |
| IP类型 | 默认公网IP | 精品原生IP或静态独享IP |
| 网络质量 | 共享带宽 | 优选本地网络运营商 |
| 计费模式 | 按月/按年 | 30天期 |
| 合规限制 | 无 | 资源创建后不支持与国内IP直连 |

## 支持地域

- 美国（洛杉矶）
- 美国（华盛顿）
- 日本（东京）
- 越南（胡志明）
- 菲律宾（马尼拉）
- 新加坡
- 英国（伦敦）

## IP分配规则

- 通过 redsocks 镜像和 CrossBorder 镜像创建的轻量应用云主机，会在公网IP地址上额外绑定一个静态独享IP。
- 通过其余应用镜像和系统镜像创建的轻量应用云主机，会直接绑定精品原生IP。
